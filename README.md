# AJAJA (코딩학원 학생 관리 시스템) 

학원에서 반복적으로 발생하는 **출석 확인**, **숙제 제출 확인**, **학습 현황 파악**의 비효율을 줄이기 위해 만든  
학생/학원용 **통합 관리 Android 앱**입니다.

---

## 📌 프로젝트 개요
- **앱 이름**: AJAJA
- **한 줄 소개**: 출석·숙제·학습리포트를 한 번에 관리하는 스마트 학원생 관리 앱
- **핵심 포인트**: 딥러닝 기반 기능을 통해 “확인/검증” 과정을 자동화하여 운영 시간을 절감

---

## ❗ 개발 동기 (학원/학생이 겪던 불편함)
- **숙제 확인의 비효율**: 제출 사진을 사람이 일일이 확인해야 해서 시간이 많이 듦
- **출석 체크의 불편함**: 단순 수기/체크 방식은 대리 출석/본인 확인이 어려움
- **학습 현황 파악 어려움**: 주차별/월별 학습량을 한눈에 보기 힘들어 피드백이 늦어짐

---

## ✅ 주요 기능
### 1) 딥러닝 기반 출석 체크 (얼굴 인식)
- 카메라로 얼굴을 인식하여 **본인 인증 기반 출석 처리**
- 저장된 학생 얼굴 특징(임베딩)과 실시간 촬영 얼굴 특징을 비교하여 동일 인물 여부 판단
- 출석 완료 시 확인 팝업 제공

### 2) 딥러닝 기반 숙제 필기 + 글씨 강도 감지
- 숙제 이미지에서 **필기 여부(감지/안됨)** 분류
- 필기된 글씨의 진하기를 **STRONG / WEAK(등급)** 으로 분류하여 학습 성실도 피드백에 활용
- 글씨가 감지되지 않으면 “제출 불가” 처리 등 제출 로직에 반영 가능

### 3) 학습 리포트 (월/전체)
- **주차별 과제 제출량(월)** 그래프 제공
- **월별 과제 제출량(전체)** 그래프 제공
- 학습 흐름을 데이터로 확인하여 상담/피드백 근거로 사용

### 4) 기본 학원 관리 기능
- 로그인/회원가입 및 사용자 정보 저장(세션/SharedPreferences)
- 공지/알림, 내 정보, 과제 제출/목록 등 화면 구성

---

## 🧠 딥러닝/AI 구현 요약
### (A) 얼굴 인식 파이프라인 (Colab)
1. 얼굴 감지 모델(MTCNN) + 얼굴 임베딩 모델(FaceNet) 로드  
2. 얼굴 영역만 crop 하는 전처리 함수 구성  
3. 학생 이미지들을 128차원 임베딩 벡터로 변환  
4. 분류기(SVM) 학습 및 모델 저장(pkl) → 앱 적용/연동

### (B) 글씨 강도 분류 파이프라인 (Colab)
1. 데이터셋 폴더 구조 확인(train/val, strong/weak/none 등)  
2. 이미지 크기/배치 등 하이퍼파라미터 설정  
3. 작은 CNN(Sequential) 모델 구성  
4. 모델 학습(train/val) 진행  
5. 정확도/손실 그래프로 학습 결과 확인  
6. 모델 저장(.h5 / TFLite 변환) 후 앱 적용
